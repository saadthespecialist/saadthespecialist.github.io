## ğŸ¨ Customization

### Update Social Links

Edit `index.html` and update the social links in the footer:
- GitHub: Update the `href` attribute for the GitHub link
- LinkedIn: Update the `href` attribute for the LinkedIn link
- Email: Update the `mailto:` link with your email address

### Update Project Information

The portfolio currently showcases the RAG from Scratch project. To add more projects:
1. Duplicate the `.project-card` section in `index.html`
2. Update the project details, description, and links
3. Add relevant technology tags

### Color Scheme

The color scheme can be customized by editing the CSS variables in `style.css`:
```css
:root {
    --bg-dark: #0a0e1a;
    --bg-card: #141824;
    --accent-primary: #3b82f6;
    --accent-secondary: #60a5fa;
    /* ... more variables */
}
```
